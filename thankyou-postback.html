How to use this page for a real test

  1. From MediaGo, go to Conversion Tracking → S2S Postback and create a conversion.
  2. Copy the postback URL template provided by MediaGo.
  3. In this file, replace POSTBACK_URL_TEMPLATE with that template and keep the {tracking_id} placeholder.
  4. Open this page with ?tracking_id=.... The page will substitute the tracking id and call the postback automatically.
